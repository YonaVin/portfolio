TEACH-IN

Online lecture hall, which contains more than 540 educational courses from the natural science faculties of Moscow State University. M. V. Lomonosov.

Project goal

Develop a design for a mobile application that would provide convenient access to lecture courses from mobile devices and provide the ability to view materials offline.

Design Highlights

Was part of this stage, an analysis of the Customer’s existing web service was carried out. During design development, the existing visual style of the site was transferred to the mobile application to maintain continuity. A design for the iPad was designed and developed separately to make using the application on the tablet even more convenient.
